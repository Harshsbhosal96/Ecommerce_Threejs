# 🛒 Ecommerce Three.js

An interactive **3D e-commerce web application** built using **Three.js** to deliver an immersive shopping experience.

---

## 🚀 Features

- 🧊 3D Product Visualization  
- 🛍️ Interactive product display  
- 🎯 Smooth camera controls  
- ⚡ Real-time rendering  
- 💻 Responsive UI  

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript  
- Three.js  
- WebGL
